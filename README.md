# 📝 Flask To-Do App

## 📌 Project Description
A simple and elegant task management app built using Flask, Bootstrap, and SQLite.

## ✅ Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Beautiful UI with background image
- Data saved using SQLite database

## 🛠️ Tech Stack
- Python
- Flask
- HTML + Bootstrap
- SQLite (via Flask-SQLAlchemy)
